# 🐶 Cédula Perruna - Editor de Identidad
Este es un proyecto divertido para crear identificaciones personalizadas para mascotas.

### ✨ ¿Qué hace?
* Genera un diseño basado en la cédula colombiana para perros.
* Permite subir la foto de la mascota.
* Crea un código QR y código de barras automático.
* Opción para imprimir el carné.
